HTML Lists and Attributes

HTML Lists are used to create a list of information in an HTML page.
HTML Lists may contain 1. Unordered Information 2. Ordered Information 3.Definitions.

2.Ordered Lists

An ordered list is used to create a list of items in sequential or numbered order.Examples are given below

  1. Tutorials
  2. Online practice
  3. Examples
  4. References
  5. videos

Attributes of ordered list:
