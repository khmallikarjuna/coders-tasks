HTML Lists and Attributes

HTML Lists are used to create a list of information in an HTML page.
HTML Lists may contain 1. Unordered Information 2. Ordered Information 3.Definitions.

3.Definition/Description Lists

A definition or description list is where the entries are arranged in format as in a dictionary or encyclopedia.
It represents a term and a relevant description in the form of lists..Examples are given below

1.Ordered list
-An ordered list is used to create a list of items in sequential or numbered order
2.Unordered list
-An unordered list is used to create a list of items which has no particular order or sequence
3.Definition list
-A definition or description list is where the entries are arranged in format as in a dictionary or encyclopedia
